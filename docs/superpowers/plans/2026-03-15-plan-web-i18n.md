# Web Frontend i18n Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Chinese + English internationalization to the entire ServerBee web frontend using react-i18next.

**Architecture:** Install i18next + react-i18next + language detector, create 7 namespace JSON files per language (en/zh), extract ~250 hardcoded strings from ~25 source files, add a language toggle in the header.

**Tech Stack:** i18next, react-i18next (>= 13.0), i18next-browser-languagedetector, TypeScript module augmentation

**Spec:** `docs/superpowers/specs/2026-03-15-web-i18n-design.md`

---

- [ ] **Step 2: Add `resolveJsonModule` to tsconfig.app.json**

In `apps/web/tsconfig.app.json`, add `"resolveJsonModule": true` to `compilerOptions`:

```json
{
```

```tsx
function LanguageSwitcher() {
  const { i18n } = useTranslation()

    </Button>
  )
}
```
